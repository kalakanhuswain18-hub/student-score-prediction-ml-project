# student-score-prediction-ml-project
A machine learning project that predicts student final exam scores based on study hours using Linear Regression. Includes data visualization, model evaluation, and prediction.
# 🎓 Student Score Prediction (Machine Learning Project)

## 📌 Overview

This project predicts students' final exam scores based on their weekly self-study hours using Machine Learning (Linear Regression).

It demonstrates the complete ML workflow:

* Data preprocessing
* Model training
* Evaluation
* Visualization
* Prediction

---

## 📊 Dataset Features

* 📚 Weekly self-study hours
* 📝 Final exam score

---

## ⚙️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📈 Model Used

* Linear Regression

---

## 📉 Model Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

---

## 📊 Visualizations

* Histogram of scores
* Scatter plot (actual vs predicted)
* Regression line

---

## 🔮 Prediction Example

The model can predict final exam score based on input study hours.

Example:

```
Study Hours: 9  
Predicted Score: 85.23
```
